📅 Calendar Program in C

A simple console-based calendar application written in C that displays the calendar for any given month and year. Ideal for beginners learning date logic and core C programming concepts.


---

✨ Features

View calendar for any month and year

Accurate leap year handling

Proper weekday alignment (Sunday–Saturday)

Lightweight, fast, and portable

Uses only standard C libraries



---

🧰 Tech Stack

Language: C

Compiler: GCC (or any standard C compiler)



---

📁 Project Structure

calendar-program/
├── calendar.c      # Source code
├── README.md       # Documentation


---

⚙️ Compilation & Execution

Compile

gcc calendar.c -o calendar

Run

./calendar

Windows:

calendar.exe


---

🧠 Program Logic Overview

Takes month and year as user input

Checks for leap year

Calculates the starting weekday of the month

Prints the formatted calendar to the terminal
